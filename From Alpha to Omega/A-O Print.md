## Verbs 
### ω-Verbs: Present Active Indicative, Infinitive, and Imperative  

|     Present     | "*release*" |    Τranslation     | *Ending: *λυ + ο/ε |
| :-------------: | :---------: | :----------------: | :----------------: |
|      S. 1       |             |     I release      |                    |
|      S. 2       |             |  You (s) release   |                    |
|      S. 3       |             | He/She/It releases |                    |
|      P. 1       |             |     We release     |                    |
|      P. 2       |             |  You (pl) release  |                    |
|      P. 3       |             |    They release    |                    |
|                 |             |                    |                    |
|      Inf.       |             |     To release     |                    |
| **Imperative:** |             |                    |                    |

|      P. 2       |             |      Release!      |                    |
|      P. 3       |             | Let them release!  |                    |
